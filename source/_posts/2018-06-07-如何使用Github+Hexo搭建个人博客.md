# 注册Github账号


 在**Github**上可以注册账号，**Github**可以作为托管的服务器运行搭建好的静态网页
 [Github官网地址](https://github.com/)
 注册**Github**账号可以让我们创建一个公共的**Repositories库**，**Node.js**和**Hexo**都基于这个库进行托管
 ~~最重要的是**Github**的个人库是完全免费的~~
